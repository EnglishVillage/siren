# SIREn Solr

## Introduction

This module provides plugins for Solr to integrate the core functionality and
the query languages of SIREn into the Solr API.

## Module Description

* **src/main/java**

    The source code of the Solr plugin.

* **src/test/java**

    The source code of the unit tests.
