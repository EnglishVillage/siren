--
This chapter covers the setup, configuration and API of the SIREn Elasticsearch plugin. The SIREn Elasticsearch plugin
provides all SIREn functionality to Elasticsearch users. Documents are indexed independently both by SIREn and
Elasticsearch so the standard Elasticsearch functionality is not modified.
